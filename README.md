## HEARTSYNC
This case study examines how data-driven analytics can enhance **HeartSync**, a global dating platform.
As the data analyst, I developed an interactive **Power BI dashboard suite** that consolidates insights across five key areas: user demographics, engagement behavior, matching & conversations, premium monetization, and safety reporting.
The dashboards provide a holistic understanding of how users interact with the app, what drives successful matches, and where opportunities exist to improve platform performance.
